# やさしいWeb制作

研装システムズ Web制作事業部の公式サイトです。Next.jsの静的サイトとしてGitHub Pagesへ公開します。

## 必要環境

- Node.js 22以上

## コマンド

- `npm run dev`: ローカル開発
- `npm run build`: `out/`へ静的出力
- `npm test`: 静的出力と主要コンテンツを検証

## 公開先

- GitHub Pages: `https://yuishoukai-ctrl.github.io/yasashii-web/`
- 独自ドメイン（DNS切替後）: `https://web.kensosystems.jp/`
